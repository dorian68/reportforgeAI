ReportForge AI

Support And Deployment Notes

This add-in is designed to profile an Excel selection and generate workbook reporting outputs plus companion artifacts for Apps Script, email, slides, and canvas-generated reporting packs.

First-Line Setup

  • Launch the add-in inside Excel Desktop or Excel on the web.
  • Select a clean range with headers when possible.
  • Use the Excel report first before enabling external integrations.

Security Notes

  • Google access tokens are session-scoped.
  • AI provider API keys are session-scoped.
  • Apps Script web app deployment is private by default.

Environment

For production hosting, set REPORTFORGE_BASE_URL during the build so the Office manifest points to the correct HTTPS origin.

Operational Guidance

For enterprise rollout, use managed Google Cloud credentials, a managed AI gateway, and validate the add-in in both Excel Desktop and Excel Web before tenant distribution.
